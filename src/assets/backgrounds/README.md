# Background Images

This folder should contain the following background images for themed pages:

## Required Images

1. **japanese.jpg** - Japanese-themed background image for the Website page
   - Recommended: Traditional Japanese architecture, cherry blossoms, or minimalist Japanese aesthetic
   - Size: 1920x1080 or larger
   - Format: JPG/PNG

2. **egyptian.jpg** - Egyptian-themed background image for the Docs page
   - Recommended: Parchment/papyrus texture, hieroglyphs, or Egyptian architectural elements
   - Size: 1920x1080 or larger
   - Format: JPG/PNG

3. **rainforest.jpg** - Rainforest-themed background image for the Video page
   - Recommended: Dense rainforest, tropical foliage, or nature scenes
   - Size: 1920x1080 or larger
   - Format: JPG/PNG

## Placeholder URLs

If images are not provided, the code will use CSS gradients and patterns as fallbacks. To use actual images:

1. Place the images in this folder with the exact names above
2. The CSS will automatically load them via `url('../../assets/backgrounds/[filename]')`

## Image Sources

You can use free images from:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

Make sure to use images that are free for commercial use.
